A) Folder Structure - top

In the "HTML" folder you will all the html files for the template. In the folder named "PSD" you will 3 PSDs for the template.

B) CSS Files and Structure - top

The template uses 4 main CSS Files. The first one is the main css file called styles.css and it is located in the same folder with the html files. In there you will find all the styles organized in sections with comments. Attention! if you change the path of this file you will have to update the paths of any background image declarations inside it.

Next you will see one more stylesheet called bootstrap.css and contains everything related to the grid. The template is based on the Twitter Bootstrap CSS Grid and you can find it's documentation on the official website here. The file mediaqueries.css containers the responsive styles needed for the template to look great on mobiles and tablets. There 6 more styles located in the colors directory. They contain all the color schemes. If you want to see them just replace the default.css with the stylesheet that you want.

C) JavaScript - top

This theme imports the following Javascript files.

  1. modernizr.js
  2. jQuery
  3. jQuery Flexslider
  4. jQuery Mobile Menu
  5. jQuery Superfish
  6. jQuery Fancybox
  7. jQuery Contact Script

All javascript files are located in the /js folder, the script that handles the infinite scrolling is at the very bottom of the index.html page.

E) Troubleshooting - top

If you load the theme from a local folder (e.g. file://..) the ajax infinite scrolling might not be working. This is due to Ajax safety issues. If you upload the theme to a live server or even Dropbox (or your localhost server) everything should be working fine.
